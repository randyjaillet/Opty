Opti

  • Opti replaces standard select elements with searchable, easily styleable, custom ones.

  • It automatically ditches the search if there aren't many options. (If you want.)

  • It supports disabled options.

  • Labels work.

  • You can have choices start off selected.

  • You can disable the whole thing.

  • It supports the multiple attribute.

  • You can even stick option groups in them.
